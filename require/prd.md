## 1. 프로젝트 개요

### 1.1 서비스 소개

고등학생 및 재수생의 **생활기록부(생기부)를 AI가 분석**하여, 입시 전략 컨설팅 리포트를 제공하는 서비스입니다. 기존 입시 컨설팅의 높은 비용(수십~수백만 원)과 접근성 문제를 해결하고, AI 기반 자동 분석 + 전문가 검수를 결합하여 합리적인 가격에 고품질 컨설팅을 제공합니다.

### 1.2 핵심 가치

- **접근성**: 5.9만~19.9만 원의 합리적 가격으로 입시 컨설팅 제공
- **신뢰성**: AI 분석 + 관리자 검수의 이중 품질 관리
- **편의성**: PDF 업로드 또는 텍스트 입력으로 간편하게 이용
- **속도**: 48시간 이내 결과 전달

### 1.3 기술 스택 (참고)

| 영역 | 기술 |
|------|------|
| 데이터베이스 | Supabase |
| AI 모델 | Gemini 2.5 Pro |
| 결제 | 토스페이먼츠 |

## 3. 유저 플로우

### 3.1 전체 흐름

```
랜딩 페이지 → 회원가입/로그인 → 생기부 제출 → (PDF일 경우) 텍스트 검증 → 플랜 선택 & 결제 → AI 분석 (백그라운드) → 관리자 검수 → 이메일로 결과 전달
```

### 3.2 단계별 상세

#### STEP 1. 랜딩 페이지

- 서비스 소개, 플랜별 가격, 분석 예시 등을 노출
- CTA: 회원가입/로그인 유도

#### STEP 2. 회원가입 / 로그인

- **이메일 + 비밀번호** 기반 인증 (소셜 로그인 미지원)
- 회원가입 시 필수 정보: 이메일, 비밀번호
- 이메일 인증 필요 여부: 추후 결정

#### STEP 3. 생활기록부 제출

두 가지 방식 지원:

| 방식 | 설명 |
|------|------|
| **PDF 업로드** | 생기부 PDF 파일을 업로드하면 라이브러리로 텍스트 자동 추출 |
| **텍스트 입력** | 생기부 내용을 직접 텍스트로 입력/붙여넣기 |

#### STEP 4. 텍스트 검증 (PDF 업로드 시)

- PDF에서 추출된 텍스트를 사용자에게 보여줌
- 사용자가 추출 결과를 확인하고 수정 가능
- 검증 완료 버튼으로 확정

#### STEP 5. 플랜 선택 & 결제

- 3가지 플랜 중 선택 (상세 구성은 섹션 4 참고)
- 토스페이먼츠를 통한 결제 진행
- 결제 완료 후 AI 분석 자동 시작

#### STEP 6. AI 분석 (백그라운드)

- 결제 완료 후 AI가 선택된 플랜에 맞는 리포트 생성
- **백그라운드 처리** — 유저는 대기하지 않음
- 유저에게는 **"48시간(이틀) 이내에 결과를 이메일로 보내드립니다"** 안내

#### STEP 7. 관리자 검수

- AI 분석 완료 시 **어드민 페이지**에서 확인 가능
- 관리자가 리포트 내용을 검수 및 필요시 수정
- 검수 완료 후 유저에게 이메일로 결과 전송

#### STEP 8. 결과 전달

- 검수 완료된 리포트를 유저의 이메일로 전송
- 전달 형식: 추후 결정 (PDF / 웹 링크 등)

## 4. 상품 구성 (플랜)

### 4.1 기본 진단형 — Lite Report (59,000원)

> **"내 상태 빠르게 진단 + 방향 잡기"**

- **예상 분량**: 10~12페이지
- **포인트**: 빠르게 내 위치 파악 + 방향 설정

| 섹션 | 페이지 | 구성 내용 |
|------|--------|-----------|
| 1️⃣ 표지 + 한줄 총평 | 1P | 핵심 요약 키워드 3개, 전체 평가 한줄 |
| 2️⃣ 전체 요약 분석 | 1P | 강점 2~3개, 약점 2~3개, 추천 전략 (학종 vs 정시) |
| 3️⃣ 세특 분석 | 3~4P | 창체 + 과목별, 강/보통/취약 분류, 사유 서술 (3~4줄 × 3개) |
| 4️⃣ 부족한 부분 분석 | 1P | 부족 영역 표시, 간단 사유 (1~2줄), 보완 활동 3개 |

## 7. 비즈니스 규칙

### 7.1 생기부 제출 규칙

- 한 번의 주문에 하나의 생기부만 제출 가능
- PDF와 텍스트 입력 중 하나만 선택
- PDF 업로드 시 반드시 텍스트 검증 단계를 거쳐야 분석 진행

### 7.2 대학 지원 조합 제공 조건

- **고2 이상 생기부**가 포함된 경우에만 대학 지원 조합 제공
- 고1 생기부만 있는 경우 대학 지원 조합 섹션 미포함

### 7.3 결과 전달 규칙

- AI 분석 완료 후 **관리자 검수 필수**
- 검수 완료 후에만 유저에게 결과 전송
- 결제 시점으로부터 **48시간(이틀) 이내** 결과 전달 목표
