# AI 리포트 평가 가이드

## 핵심 원칙

입시는 단순히 내신 등급이나 정량적 지표로 결정되지 않는다. 5등급제 도입 이후 등급 간 변별력이 약해지면서 생기부의 정성적 평가 비중이 확대되고 있다. AI는 단순히 성적이나 결과를 해석하는 것이 아니라, 생기부에 나타난 학업 과정, 선택의 이유, 활동의 맥락을 중심으로 평가를 수행해야 한다.

---

## 대학의 4가지 평가 축

### 1. 전공적합성

- 단순 "이 전공을 희망한다"는 진술이 아님
- 선택 과목, 세특, 창의적 체험활동, 탐구활동 등이 하나의 전공 방향으로 일관되게 이어지는지 판단
- AI는 과목 선택 → 세특 → 탐구 → 진로의 연결 구조를 파악해야 함

### 2. 학업역량

- 성적 자체보다 수업 내 이해도, 개념 적용 능력, 심화 학습 여부 기반 평가
- 주로 세특의 서술 방식과 내용의 깊이를 통해 드러남
- AI는 세특에서 개념 이해 → 적용 → 확장 흐름이 있는지 분석해야 함

### 3. 탐구역량

- 단순 참여가 아닌 전체 흐름 평가:
  - 문제 설정 → 탐구 과정 → 결과 도출 → 확장 또는 한계 인식
- AI는 탐구의 깊이와 확장성을 평가해야 함

### 4. 발전가능성

- 활동들이 시간 흐름에 따라 점점 심화되고 구체화되는지 평가
- 성공 경험뿐 아니라 시행착오, 한계 인식, 보완 시도가 오히려 긍정적
- AI는 성장의 흔적과 변화의 과정까지 함께 평가해야 함

---

## 우수한 생기부의 구조

**이상적인 구조:**
과목 선택 → 세특 내용 → 탐구 활동 → 진로 방향이 하나의 논리적 흐름으로 연결

**높은 평가 요소:**

- 스스로 학습을 확장하고 방향성을 설정해 나간 흔적
- 개념 이해 → 적용 → 확장의 흐름이 드러나는 서술
- 구체적인 행동과 사고가 드러난 기록
- 학년별 점진적 심화 (1학년 탐색 → 2학년 구체화 → 3학년 심화)

**감점 요인:**

- 단순 참여 위주의 활동
- 반복적이고 추상적인 표현 (예: "성실함", "적극적 참여")
- 전공과 무관한 활동의 과다
- 활동 간 연결성 부족
- "왜 이 활동을 했는지"에 대한 맥락 부재

---

## 과목 선택 전략 평가

- 학생이 쉬운 과목 위주로 안정적인 성적을 받았는지 vs 진로 관련 과목을 선택하고 도전적 학습을 했는지 구분
- 전공과의 연관성, 선택의 일관성, 난이도에 대한 도전 여부 평가
- 특정 전공 관련 핵심 과목 이수 여부를 중요하게 반영

---

## 세특(세부능력 및 특기사항) 분석 기준

- 단순 키워드 추출이 아닌, 문장 구조 기반 분석:
  - 탐구 과정이 존재하는지
  - 사고의 확장이 있는지
  - 구체적인 사례가 포함되어 있는지
- 결과만 적힌 기록보다 **과정이 드러나는 서술**을 높게 평가
- 짧은 문장이라도 구체적 행동과 사고가 드러나면 높은 평가

---

## 교과 성적 해석 (다차원 분석)

### 원점수 + 평균 + 표준편차 + 수강자 수 종합 분석

동일한 등급이라도 실질적 가치가 다름:

| 지표                        | 해석                                                        |
| --------------------------- | ----------------------------------------------------------- |
| **표준편차 큼**             | 시험 변별력 높음 → 상위권 성적의 가치 높음                  |
| **표준편차 작음**           | 점수 분포 촘촘 → 등급 간 실질 차이 작음                     |
| **수강자 수 많음**          | 경쟁 집단 크고 평가 신뢰도 높음 → 동일 등급이라도 높은 평가 |
| **수강자 수 적음**          | 경쟁 구조 제한적 → 전공 연관성과 함께 맥락적 해석 필요      |
| **평균 낮고 표준편차 큼**   | 어려운 시험에서의 높은 성적 → 상위권 경쟁력 의미            |
| **평균 높고 표준편차 작음** | 쉬운 시험 → 높은 등급이라도 변별력 낮음                     |

### AI 분석 방향

- 등급을 그대로 해석하지 않고, 원점수가 평균 대비 어느 위치인지(상위 몇 %) 분석
- 점수 분포 내 위치, 시험 난이도 종합 분석
- 과목 난이도, 선택 맥락, 점수 분포, 경쟁 규모를 종합 고려하여 실제 의미 해석

## 활동 평가 원칙

### 양보다 밀도와 차별성

- 많은 활동 나열보다 몇 개의 활동을 깊이 있게 탐구한 것이 높은 평가
- 흔한 주제나 형식적 활동보다 자신만의 문제의식이나 관점 기반 탐구가 중요
- AI는 활동 개수를 긍정적으로 해석하기보다, 깊이와 차별화 요소 중심으로 평가

### 일관성과 스토리라인

- 생기부는 하나의 이야기처럼 읽히는 것이 이상적
- 학년이 올라갈수록: 관심 분야 구체화 → 활동 수준 심화 → 진로 방향 명확화
- 방향 변화 시 변화에 대한 이유와 맥락이 설명 가능해야 함
- AI는 시간 흐름에 따른 변화, 활동 간 연결성, 방향성의 일관성을 종합 판단

---

## 사정관 관점의 핵심 포인트

### 오해와 현실

| 오해                              | 현실                                                       |
| --------------------------------- | ---------------------------------------------------------- |
| 좋아 보이는 활동을 많이 하면 유리 | 활동의 학업 맥락 속 의미가 더 중요                         |
| 세특은 결과 기록                  | 세특은 과정 기록 (접근 방식, 사고 과정, 변화)              |
| 처음부터 완벽한 전공 일관성 필요  | 탐색 → 구체화 → 심화의 자연스러운 성장 과정이 더 높은 평가 |
| 성공 경험만 중요                  | 시행착오, 한계 인식, 보완 시도도 긍정적                    |

### 기록의 신뢰도와 진정성

- 과장된 표현이나 형식적 문장은 사정관이 쉽게 구분
- "심화 탐구함", "비판적으로 사고함" 등 반복되지만 구체적 내용 없는 경우 → 형식적 기록으로 판단
- 짧은 문장이라도 구체적 행동과 사고가 드러나면 훨씬 높은 평가

### 최종 판단 기준: 발전가능성

- "이 학생이 대학에 와서도 계속 성장할 수 있는가"
- 스스로 질문을 만들고, 학습을 확장하며, 실패를 통해 개선해 나가는 경험 유무

## 생기부 작성 전략 (학생/학부모 가이드)

### 핵심 이해

- 생기부는 "활동 기록"이 아니라 **평가 자료**
- 핵심은 기록되는 순간이 아니라 **기록되기 전 과정 설계**

### 기록 가능한 행동 만들기

- 단순 참여가 아니라 질문, 발표, 토론, 추가 탐구 등 교사가 기록할 수 있는 행동을 의도적으로 수행
- 수업에서 배운 개념 바탕으로 추가 자료 탐색, 수행평가 확장, 교과 내용과 관심 분야 연결

### 기록 가능 vs 불가능 구분

| 기록되기 어려운 행동      | 기록 가능한 행동                   |
| ------------------------- | ---------------------------------- |
| 혼자 공부, 단순 문제 풀기 | 발표, 보고서 작성, 토론 참여       |
| 외부 스펙, 대외활동       | 학교 내 수업과 평가 과정 기반 활동 |

### 학년별 전략

| 학년  | 전략                                       |
| ----- | ------------------------------------------ |
| 1학년 | 다양한 과목 경험, 관심 분야 탐색           |
| 2학년 | 전공 방향 설정, 관련 과목 선택과 탐구 시작 |
| 3학년 | 정리 및 완성도 향상                        |
